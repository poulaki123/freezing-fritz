The outside temperature is given as words ("Freezing", "Hot", etc.) so first those are mapped to actual numbers. Outliers are removed, and two custom features are engineered — combining door and window states to capture airflow effects. Several regression models are then trained and compared (Gaussian Process, Ridge, ElasticNet, Gradient Boosting), and finally the three best are combined into a Voting Regressor ensemble to get the most accurate temperature prediction.
